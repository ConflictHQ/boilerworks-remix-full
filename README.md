# Boilerworks Remix Full

> Edge template -- Remix on Cloudflare Workers. Server-first React at the edge.

**Status:** Planned

Remix full-stack edge application on Cloudflare Workers. Server-first React with nested routing, progressive enhancement, and web standard APIs. Loaders and actions for data flow, no client-side state management needed. Ideal for React teams that want server-first architecture at the edge.

**Edge template** -- production deployment targets Cloudflare Workers, not Docker.
